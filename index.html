Student Performance Prediction

A student performance prediction system uses boosting models like CatBoost and Gradient Boost to achieve 96% accuracy. After preprocessing, mutual information is applied for feature selection. SHAP (SHapley Additive exPlanations) is then used to interpret the model's predictions, showing how each feature influences the outcome. Partial Dependence Plots (PDP) visualize the relationship between key features and predicted performance, offering deeper insights into the factors driving student success.

Automated sitting management system

An automated sitting arrangement system uses Python and the OpenPyXL library to assign seating to students based on a predefined room structure in an Excel sheet. The system reads student data from a CSV file, including details like roll numbers and names. It then maps these students to the available seats in the room, as defined in the Excel template, ensuring the seating arrangement follows any specified rules or constraints. Once the arrangement is complete, the updated Excel sheet is automatically generated and provided for download, offering a seamless and efficient seating plan distribution.
